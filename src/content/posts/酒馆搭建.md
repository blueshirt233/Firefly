# 酒馆搭建指南

个人的逼逼赖赖：
{我在b站上也看到过在windows上一键搭建酒馆的方法，但是电脑我晚上都是要关掉的，而我想要24小时都运行（因为电脑在我房间里，一直开着电脑风扇声音太大晚上吵着我睡觉），就想用小型的工控机做服务器声音很小，网络上找了很多教程，但是跟我想要的有点不一样，就打算自己做一个好了}
声明一下，由于本人没有公网ip，又不想花钱买云服务器（工控机是毛的），所以就只能在家里用，出去就不行了。
我的系统是rockylinux9.7，配ip和怎么安装我就先跳过了，基础的部分就先B站上自己速成一个就行，安装好后先关闭防火墙和selinux，并且更新一下。
~~~bash
dnf update -y
~~~
## 安装环境

安装Node.js 20
~~~bash
curl -fsSL https://rpm.nodesource.com/setup_20.x | sudo -E bash -
sudo dnf install -y nodejs
~~~
验证版本
~~~bash
node --version
npm --version
~~~
## 安装酒馆
~~~bash
cd /root
git clone https://gitee.com/boomer001/SillyTavern.git #这个是从gitee上面随便找的镜像，镜像不一定实时更新，要是你能科学上网那可以去github上面去拿
cd SillyTavern
~~~
设置npm镜像
~~~bash
npm config set registry https://registry.npmmirror.com
~~~
安装依赖
~~~bash
npm cache clean --force
npm install --legacy-peer-deps
~~~
测试启动
~~~bash
./start.sh
~~~
## 持久化配置
安装PM2
~~~bash
npm install -g pm2
~~~
启动服务
~~~bash
pm2 start server.js --name sillytavern
~~~
停止服务
~~~bash
pm2 stop sillytavern
~~~
删除服务
~~~bash
pm2 delete sillytavern
~~~
需要更加具体的操作命令就百度或者问ai
## 配置酒馆
~~~bash
vi /root/SillyTavern/config.yaml
~~~
具体修改位置，直接复制到文本文档里ctrl+f搜索就行
~~~
listen: false
改
listen: true

port: 8000
这个改端口号，不要冲突就行，默认8000

whitelist:
  - ::1
  - 127.0.0.1
 这里要把你当前的网段或者ip增加到白名单里面去（一般是网段，现在家用都是自动获取，也不用这么麻烦）
 举例：
 whitelist:
  - ::1
  - 127.0.0.1
  - 192.168.1.0/24
要改哪个网段要看你当前服务器和客户机在哪个网段里，最好都加到白名单里面去
~~~
现在可以随便拿一台手机或电脑连到同一个网段里,浏览器网址输入{服务器ip：端口号}就可以进入
我了解到还可以配置登录密码和多用户登陆的设置，目前的家里就我一个人在用也用不到这个设置就没在意，需要的百度一下。
